Lilac · 7GUIs

7GUIs is a benchmark of 7 small GUI tasks that exercise the typical pain points of UI frameworks. This gallery is a Lilac implementation, built with lilac-cli from components/gallery-nav.lil + plain HTML pages with inline <script type="text/ruby">.

  1. 1
    Counter

    Basic signal + button.

  2. 2
    Temperature Converter

    Two-way binding between two inputs via computed.

  3. 3
    Flight Booker

    Dependent inputs, validation, enabled/disabled submit.

  4. 4
    Timer

    Effect-driven setInterval + progress bar + adjustable duration.

  5. 5
    CRUD

    Filter, list, create / update / delete with bind_list.

  6. 6
    Circle Drawer — TBD

    Canvas / SVG + undo / redo. Not yet implemented.

  7. 7
    Cells — TBD

    Spreadsheet with formula parsing and dependency graph. Not yet implemented.
