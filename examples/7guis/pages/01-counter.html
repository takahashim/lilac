How it works

Reactive 状態を signal(0) で 1 行作り、ボタンの data-on-click がインクリメントを呼ぶだけ。data-text が Signal 変更を自動で DOM に反映する。

  • signal(value) — 書き換え可能な reactive cell
  • Signal#update {|n| n + 1 } — 関数で更新(競合回避)
  • data-on-click="inc" — メソッド名指定だけで bind、refs 不要
  • data-text="@count" — Signal の値を textContent に流す

One line of reactive state via signal(0), and the button's data-on-click calls the increment. data-text automatically syncs Signal changes to the DOM.

  • signal(value) — writable reactive cell
  • Signal#update {|n| n + 1 } — functional update (avoid races)
  • data-on-click="inc" — bind by method name, no refs needed
  • data-text="@count" — pushes the Signal value into textContent
