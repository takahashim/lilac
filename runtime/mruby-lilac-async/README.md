# mruby-lilac-async

Optional async extensions for `mruby-lilac`.

Provides:

- `Fetchy`
- `Lilac::Selector`
- `Lilac::Resource`
- `Lilac::Component#selector`
- `Lilac::Component#resource`

Load this gem when your Lilac app needs async data loading or
selection helpers on top of the core component/runtime layer.
